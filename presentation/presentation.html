"You'll be licking your fingers by the time you're done"

Initial Design

Users were mainly interested in academic features:

  • Syllabus
  • Course Materials
  • Assignment Turn-in

Goal: Achieve these academic tasks with high efficiency and learnability

Under the Hood

  1. Python Django framework to process data scraped from my.rochester.edu
  2. Beautiful Soup scraping/parsing library for Python
  3. "Parsley" responds to the extension with JSON

On the Surface

  1. JavaScript to load parsed content into the extension
  2. Building the interface:
    • User freedom: Add new content, show/hide resources
    • Error reduction: Large buttons, show/hide available only in edit mode
    • Visibility: Bright colors and high contrast
    • Learnability: Uses familiar flow from Blackboard
