Designed and presented by:
Trevor Filer, Chelsea Flint and Ian Wilson

"You'll be licking your fingers by the time you're done"

Initial Design

Users were mainly interested in academic features:

- Syllabus
- Course Materials
- Assignment Turn-in

Goal: Achieve these academic tasks with high efficiency and learnability

Under the Hood

1. Python Django framework to store data scraped from my.rochester.edu

2. Beautiful Soup scraping/parsing library for use with Django

3. Django app "Parsley" to implement Beautiful Soup with the Django framework

On the Surface

1. JavaScript to load Django database content into the extension

2. Building the interface:

- User freedom: Add new content and show/hide
- Error reduction: Large course button, small hide button
- Visibility: Colors - what catches the eye
- Learnability: Point-and-click design with clear indicators for features
